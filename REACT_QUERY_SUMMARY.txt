WHAT IS REACT QUERY?
================================================================================

React Query (TanStack Query) is a data fetching and caching library that:

✅ Automatically caches data
✅ Prevents page reloads on data updates
✅ Deduplicates API requests
✅ Handles background refetching
✅ Manages loading and error states
✅ Provides optimistic updates
✅ Automatically garbage collects unused data

RESULT: 50-70% fewer API calls, faster pages, better UX!

================================================================================
USAGE EXAMPLES
================================================================================

EXAMPLE 1: Fetch Classes
Before:
  const [classes, setClasses] = useState([]);
  useEffect(() => {
    fetch('/api/classes').then(r => r.json()).then(d => setClasses(d.data));
  }, []);

After:
  const { data: classes = [] } = useClasses();

EXAMPLE 2: Create Class
Before:
  const handleCreate = async (data) => {
    await fetch('/api/classes', { method: 'POST', body: JSON.stringify(data) });
    window.location.reload();  // Page reload!
  };

After:
  const { mutate: createClass } = useCreateClass();
  const handleCreate = (data) => {
    createClass(data);  // No page reload!
  };

EXAMPLE 3: Update Class
Before:
  const handleUpdate = async (id, data) => {
    await fetch(`/api/classes/${id}`, { method: 'PUT', body: JSON.stringify(data) });
    window.location.reload();  // Page reload!
  };

After:
  const { mutate: updateClass } = useUpdateClass();
  const handleUpdate = (id, data) => {
    updateClass({ classId: id, data });  // No page reload!
  };

================================================================================

Retry Logic:

  refetchOnMount: false        // Don't refetch on component mount
